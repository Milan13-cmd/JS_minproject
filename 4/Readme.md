# Text Div Image Toggle

## Description

This project involves creating a dynamic interaction where an image appears when the mouse enters a text div, disappears when moving out of the text, and becomes visible again when moving out of the div.

## Features

- **Mouse Enter/Leave Interaction:** Image appears on mouse enter and disappears on mouse leave within the text div.
- **Visibility Toggle:** Image becomes visible again when moving out of the text div.
- **User-Friendly:** Provide a seamless and intuitive user experience.
